# AndroidCommon

android base library

## 引入方式

#### gradle

    compile 'me.codeboy.android:android-common:1.0.1'

#### maven

     <dependency>
         <groupId>me.codeboy.android</groupId>
         <artifactId>android-common</artifactId>
         <version>1.0.1</version>
     </dependency>
